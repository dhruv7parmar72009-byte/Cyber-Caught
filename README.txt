Cyber Caught - Offline Demo

Files:
- index.html
- styles.css
- app.js

How to run:
1. Open index.html in a browser.
2. Create a Gmail account in the demo login form.
3. Upload an audio file or use the live microphone button.
4. Use the Download PDF Report button to export the scan.

Notes:
- The project works without internet.
- Charts and PDF export are implemented locally.
